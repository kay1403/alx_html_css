# CSS Advanced Project

## Project Overview
This project is part of the ALX Front-End Web Development curriculum, focusing on advanced CSS concepts and responsive web design. The goal is to recreate a landing page for "SmileSchool" based on a provided Figma design, implementing proper styling, layout techniques, and responsive behavior.

## Learning Objectives
- Understand CSS fundamentals and how to apply styles to HTML elements
- Work with CSS classes and selectors effectively
- Compute CSS specificity values
- Implement CSS box model properties
- Understand how browsers load webpages
- Create responsive designs that work across different devices

## Requirements
- All files must end with a new line
- W3C compliant code that validates with W3C-Validator
- No external libraries or frameworks (pure HTML/CSS/JavaScript)
- Responsive design implementation
- Accurate recreation of the Figma design
